How to use — 2 minutes

  1. Get your Warcraft Logs API key (once): scroll down on your WCL profile, enter consumes as "V1 Client Name", hit Set, copy the "V1 Client Key" into the key field. It's remembered by your browser.
  2. Pick the game version — WoW Classic Anniversary or Classic Era — and paste your raid's log URL (or just the report code).
  3. Hit Analyze log. The tool pulls every tracked consumable (flasks, elixirs, potions, bombs, scrolls, runes …) — takes about a minute. Flasks are counted wipe-safe.
  4. Prices: the TSM source (realm · region · faction) is detected from the log automatically. Hit Update TSM prices now for the freshest data, or edit any price by hand.
  5. Share it: export the CSV, download the table as a PNG image (with gold costs or — when gold is hidden — coverage %), copy Discord markdown, or paste a Discord webhook URL (Server Settings → Integrations → Webhooks) and hit Send — you get a branded embed with a rendered chart and two random fun facts.
  6. History: every analysis is stored in your browser. Filter by all time / last month / single reports / custom range and copy a summary as markdown.
  7. Custom consumables: anything missing? Add it with its exact WCL name + TSM item ID — detection (cast vs. buff) is automatic.

🔒 Everything — API key, webhook, history, custom items — lives only in your browser (localStorage). Requests go exclusively to warcraftlogs.com, discord.com and the price feed.

Counting method: short buffs & potions via cast events (exact); flasks (120 min, persist through death — wipes never double-count), elixirs (60 min) and scrolls (30 min) are estimated from buff uptime bands against fight boundaries — estimates, since logs only see combat time. Drums: cost per use = item price / 50 charges. Mana Emeralds & Healthstones count as 0 g (conjured).
